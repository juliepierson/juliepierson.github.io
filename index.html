Mes projets

Une collection hétéroclite de projets, ayant trait pour la plupart à la géomatique, fonctionnels ou non, plus ou moins mis à jour et documentés...

Cartes D3

Une bibliothèque de cartes avec D3 (choroplèthes, cercles proportionnels, anamorphoses) avec zoom et pan + interaction via listes déroulantes ou autre.

Ces cartes sont réparties dans deux projets : D3_produits_terroir et D3_volontaires.

Interface de saisie de données Leaflet

saisie_leaflet : interface hors ligne de saisie de points sur un raster avec Leaflet; les données saisies peuvent être exportées au format geojson pour être exploitées dans un logiciel SIG.

DissolveWithStats

DissolveWithStats est un plugin QGIS pour regrouper des entités ayant la même valeur pour un attribut, et calculer des statistiques sur les autres attributs.

Ce plugin n'est plus utilisable actuellement, étant basé sur un outil QGIS dont le fonctionnement a changé, il me manque un peu de temps pour le mettre à jour...
